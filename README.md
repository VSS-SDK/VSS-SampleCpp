Installation
------------
[INSTALL][install]

Cloning
-------
```
$ git clone --recursive https://github.com/SIRLab/VSS-SampleStrategy.git
```


Development and Use
-------------------

```
$ make proto
$ make 
$ ./VSS-SampleCpp -c [yellow || blue]
```
